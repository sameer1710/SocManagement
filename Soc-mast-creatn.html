As a first Step to set up your Complex, create blocks of your Apartment Complex below .PLease Note:

  • Blocks are also referred as Towers / Buildings in some apartment Complexes.
  • Even incase your complex does not have blocks, you need to create atleast one block (you can call the block "Flat").
  • Do note that each block name must be different and should be less than 30 characters in length.
  • If you have multiple wings in each tower - create one block per tower-wing combination. For example. Tower 1 - A wing, Tower 1 - B wing, etc...
  • You can always add more blocks here.
 

As next steps, please enter the unit numbers that you would like to add for each block.

Please not:

  • Units are also referred as flats, apartments, villas, plots, etc...
  • Separate the unit numbers by commas or enter unit numbers on separate lines.
  • Do note that each unit number must be different and cannot be more than 6 characters.
  • You can add all units present in each block in a single step below. If you do not have all details, you can always add more units later.
  • You will not be able to edit the unit number once created. If you want some unit number changed, please send a mail to support@zit.com


For example enter 001, 002, 003, 004, 101, 102, 103, 104 to create 8 units
